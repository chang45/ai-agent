# 天气决策助手

这是一个基于豆包兼容 OpenAI 接口的 Python 小项目，支持：

- 查询当前公网 IP 对应的大致位置
- 查询指定城市的天气信息，并辅助判断是否适合晒被子、旅游、出门等
- 在用户明确需要时，提供未来 7 天的天气预报分析
- 通过工具调用让模型自动选择查询位置或天气

## 功能说明

### 1. 当前位置查询
使用百度地图 IP 定位接口，根据当前公网 IP 返回大致位置。

### 2. 天气决策辅助
使用百度地图地理编码获取城市坐标，再调用和风天气接口查询天气数据，帮助用户判断今天是否适合晒被子、旅游、出门、晾衣服等。

### 3. 7 日天气分析
当用户明确询问“未来几天”“7天”“这周”时，助手会基于 7 天天气数据做趋势分析。

### 4. 模型工具调用
模型可根据用户问题自动决定是否调用：

- `get_location`
- `get_weather`

## 项目文件说明

- `main.py`：程序入口，负责与模型对话和工具调用编排
- `tool_define.py`：工具定义，描述模型可调用的工具
- `tool_func.py`：工具实现，负责位置和天气查询
- `system_prompt.py`：系统提示词，约束助手回答风格
- `.env`：本地环境变量配置文件（**不要上传到 GitHub**）

## 安装依赖

先安装依赖：

```bash
pip install openai python-dotenv requests
```

## 环境变量配置

本项目依赖本地 `.env` 文件读取密钥。

请在项目根目录手动创建 `.env` 文件，并写入以下内容：

```env
ARK_API_KEY=你的豆包APIKey
ARK_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
ARK_MODEL=你的模型ID
BAIDU_MAP_KEY=你的百度地图AK
QWEATHER_KEY=你的和风天气Key
```

### 说明

- `ARK_API_KEY`：豆包/火山方舟接口密钥
- `ARK_BASE_URL`：豆包兼容 OpenAI 的接口地址
- `ARK_MODEL`：你使用的模型 ID
- `BAIDU_MAP_KEY`：百度地图开放平台 AK
- `QWEATHER_KEY`：和风天气 API Key

## 重要提醒

`.env` 文件包含你的私密密钥，**不要提交到 GitHub**。

建议在 `.gitignore` 中加入：

```gitignore
.env
```

如果你已经误提交过密钥：

1. 立即去对应平台重置密钥
2. 修改本地 `.env`
3. 不要再把真实密钥写进代码文件中

## 返回结果说明

### 位置查询
会返回：

- 地址
- 经纬度
- IP
- 省市区街道信息

### 天气查询
会返回或用于分析：

- 城市
- 每日天气状况
- 最高温 / 最低温
- 风力
- 湿度
- 更新时间
- 是否适合某项活动的判断与建议

## 常见问题

### 1. 百度定位失败
请检查：

- `BAIDU_MAP_KEY` 是否正确
- 百度地图接口是否已开通
- AK 是否有权限调用 IP 定位接口

### 2. 和风天气失败
请检查：

- `QWEATHER_KEY` 是否正确
- Key 是否有 7 日天气接口权限
- 查询坐标格式是否正确

### 3. 模型可以聊天但工具不能用
说明通常是：

- `ARK_*` 配置正确
- 但百度地图或和风天气 key 配置有误
